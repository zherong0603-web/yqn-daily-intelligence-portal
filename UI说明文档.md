# UI 说明文档

## 产品定位

YQN Growth War Room 是一个每日增长情报、MQL 质量、内容实验和管理层摘要的自动化门户。它不是普通工程 dashboard，而是面向老板、增长、销售和内容团队的全球数字化物流情报终端。

## 视觉语言

- 主底色：黑色 / 深蓝科技底。
- 主品牌色：YQN 蓝。
- 行动提示：金黄色。
- 风险提示：橙色 / 红色。
- 业务图形：全球航线、仓网节点、集装箱纹理、物流雷达。
- 语义关键词：美国仓、北美仓网、跨境电商履约、OMS、WMS、BI、头仓配链路。

## 信息架构

顶部导航控制在 6 个入口：

1. 今日
2. 老板摘要
3. MQL
4. 实验
5. 历史
6. 配置

## 首页

首页首屏分三层：

1. 一句话定位和当前状态。
2. 今天先看这 3 件事。
3. 两个主按钮：看管理层摘要、打开配置向导。

如果是 Demo 样例，首屏会醒目提示“不建议作为正式日报发给管理层”。

## 配置向导

`/setup/` 是非技术用户的 3 分钟配置中心。它包含：

- 5 步进度条。
- 配置状态面板。
- OpenAI API Key 操作卡。
- OpenAI Model 操作卡。
- 飞书机器人操作卡。
- 加密密码操作卡。
- 手动测试一次操作卡。

## 老板摘要

`/boss/` 只保留 5 块：

1. 今日结论。
2. 今天最大异常。
3. 需要老板关注。
4. 今天推进动作。
5. 复制和打印按钮。

## 移动端

手机端首页优先显示：

1. 当前模式。
2. 今日一句话结论。
3. 今天先看 3 件事。
4. 看老板摘要按钮。
5. 打开配置向导按钮。
6. 今日行动清单。

其他模块默认折叠，避免首屏堆满小字。
