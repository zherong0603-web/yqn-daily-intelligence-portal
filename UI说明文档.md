# UI说明文档

## 产品定位

V3 页面已经从普通日报门户升级为 `YQN Growth War Room`：面向老板、增长负责人、内容负责人和执行者的商业情报作战台。

## 设计语言

- 深色底：降低长时间阅读疲劳。
- YQN 蓝：用于主按钮、信号线、导航和关键状态。
- 金黄色：用于老板关注、打印/分享、风险提示和强提醒。
- 卡片半径：统一 8px，保持商务工具感。
- 航线/仓网隐喻：首屏有信号路线视觉，表达物流、仓网、情报节点。

## 页面结构

- 首页：今日作战台、今天先看 3 件事、管理层摘要、MQL、组织缺口、内容实验、个人日报、历史归档、搜索、安全状态。
- 管理层摘要：老板 30 秒看懂今天，支持复制老板版摘要。
- 报告页：摘要、可折叠情报卡片、行动清单、来源、上一天/下一天、复制、打印。
- 归档页：全部历史、月归档、周归档、日历视图。

## 关键交互

- 复制分享链接。
- 复制老板版摘要。
- 复制今日行动清单。
- 打印 / 保存 PDF。
- 搜索关键词。
- 主题筛选。
- 按日期、按月、按周查看。
- 加密锁定 / 解锁。

## 移动端

手机端采用单列布局，导航、按钮、卡片和报告内容不会横向溢出。已生成手机首页和手机报告页 fullPage 截图。

## 安全提示

页面明确提示 GitHub Pages 是公开网页，`noindex` 和 `robots.txt` 不是访问控制。加密模式是客户端加密，不是企业级登录。
