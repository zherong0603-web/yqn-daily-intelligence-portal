# YQN 共享知识源

这里保存晨报机器人、Codex 本机项目和 ChatGPT Project 可以共同引用的稳定知识。

- `yqn-capabilities.yaml`：YQN 可公开表达的能力与禁止自行补全的边界。
- `../config/north-america-intelligence.yaml`：美国、墨西哥和美墨联动的官方信息源、搜索范围、价值评分与 2+2+1 配比。

使用规则：

1. ChatGPT Project 负责对话和项目记忆，这些文件负责保存可验证的事实。
2. YQN 官网只校准“我们能做什么”，不能冒充外部新闻来源。
3. 客户、报价、合同、毛利、账号、密钥和未公开仓库数据不得写入这里。
4. 修改内部确认口径前，必须由业务负责人再次确认。
