## 还不能自动完成的事

- `OPENAI_API_KEY`：必须由用户在 GitHub Secrets 手动填写，因为这是 OpenAI API 私密密钥，不能发到聊天里，也不能写进仓库。
- `OPENAI_MODEL`：必须由用户在 GitHub Variables 手动填写，因为 Codex 不能确认用户 OpenAI API 账号实际可用的模型名。
- `FEISHU_WEBHOOK_URL`：如果要飞书通知，必须由用户在 GitHub Secrets 手动填写，因为这是飞书机器人的私密推送地址。
- `PAGE_ACCESS_PASSPHRASE`：如果要开启加密，必须由用户在 GitHub Secrets 手动填写，因为这是页面访问密码，不能由 Codex 代填。
