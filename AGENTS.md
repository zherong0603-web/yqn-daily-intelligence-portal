# 项目工作准则

本仓库的主产品是 `YQN 每日重点简报`。

## 产品边界

- 不把首页做成增长作战台。
- 不把首页做成线索质量看板。
- 不把首页做成工程说明页。
- 不把按钮写成对象化分享文案。
- 旧 `/boss/`、`/executive/` 只做兼容入口，不进主导航。

## 默认内容范围

优先：

- 美国仓
- 跨境物流
- 小红书获客
- AI 自动化工具
- 内容效率
- 销售辅助

默认不做：

- 个人创业机会
- 个人副业信号
- 泛商业鸡汤
- 和 YQN 无关的热点汇总

## 交付产物位置

本次对话的截图、录屏、zip 统一放到：

`/Users/wzr/Desktop/Codex-本次对话-YQN简报/`

不要把新截图和录屏散落到其他桌面文件夹。

## 安全规则

- 不要求用户把密钥发在聊天里。
- 只读取 GitHub Secrets / Variables 的配置状态。
- 不在日志、HTML、JSON、README、artifact、飞书卡片里泄露 secret。
- GitHub Pages 是公开网页，必须保留 noindex 和 robots.txt。
- 加密模式不是企业级登录系统。
