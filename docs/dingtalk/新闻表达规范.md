# 新闻表达规范

每条信号必须按事实、影响、业务转化、动作拆开写。

写法要求：
- 倒金字塔：先写最重要事实。
- 5W1H：明确谁、何时、发生什么、为什么影响。
- 发生了什么：只写事实，不写判断。
- 为什么重要：只写影响，不写解决方案。
- YQN 可用点：只写业务转化。
- 今天动作：只写当天可执行动作。

必须有：
- source_name
- source_url
- source_published_at；如果来源未写日期，填“来源未注明日期”
- collected_at
- info_region
- info_type
- confidence_label：高 / 中 / 低
- is_test_data
- source_summary

禁止空话：
- 持续关注
- 提升效率
- 加强学习
- 赋能业务
- 值得重视
- 市场变化明显

禁止把模型判断伪装成事实。
