# 内容口径说明

V1.1 的晨报不是单一仓配摘要，而是跨境增长情报。

五个模块：
1. 市场与政策：美国、欧洲、中国跨境政策、关税、清关、低值包裹、贸易监管、国内跨境出口环境。
2. 平台与卖家：Amazon、TikTok Shop、Walmart、eBay、Temu、Shein、Shopify 的公开卖家规则和经营痛点。
3. 竞品与履约：Amazon Supply Chain、菜鸟、4PX、万邑通、FedEx、UPS、USPS、北美 3PL、头程、海外仓、尾程、退货。
4. 增长与线索：小红书 B2B 获客、钉钉私域、销售承接、MQL 质量、线索分层、内容选题。
5. YQN 今日可用动作：内容、销售、履约、数据、FAQ、话术和证据点。

群内只展示前 4 个模块，归档页展示完整 5 模块。

demo 模式是公开来源口径样例，不代表实时新闻。live 模式如果回退到样例数据，也必须保留测试标识。
