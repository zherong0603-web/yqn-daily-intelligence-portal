# 内容口径说明

V1.4 的晨报定位是“YQN 北美情报晨报”：让老板、运营、销售、内容和履约同事快速判断，哪些美国、墨西哥和美墨跨境变化会影响卖家的成本、时效、合规或库存。

固定配比：
1. 两条美国或美仓信号。
2. 两条墨西哥或墨仓信号。
3. 一条美墨联动信号。

群内和归档页都展示完整 5 条信号。

筛选原则：
- 每次必须真实搜索三组官方来源，重要政策回到政府、平台、港口或承运商原页复核。
- 每条按 100 分评估卖家损益、紧迫性、YQN 链路相关性、商家覆盖、来源可信度和新变化；70 分以下不进群。
- 每条只回答发生了什么、何时生效、影响谁、影响哪段链路、卖家检查什么、YQN 能承接什么。
- YQN 能力只引用官网公开或内部确认可公开的口径；未知数据明确写“尚未确认”。
- 不为发送而发送：数量、配比、来源或搜索不达标时直接停止发送。

demo 模式是公开来源口径样例，不代表实时新闻。live 模式不使用 demo 样例充当真实内容；模型不可用时只基于真实候选源生成保守版。
