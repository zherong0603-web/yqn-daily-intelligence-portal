# 内容口径说明

V1.2 的晨报定位是“YQN 每日 5 分钟”：让老板、运营、销售、内容和履约同事都能快速知道今天最值得关注的公开变化。

五个模块：
1. 行业市场：政策、关税、清关、跨境出口环境、美国消费和卖家经营变化。
2. 平台变化：Amazon、TikTok Shop、Walmart、eBay、Shopify 等平台规则、流量、履约和售后变化。
3. 客户需求：客户正在问什么、MQL 质量、内容获客、线索承接和销售首问变化。
4. 履约供给：北美 3PL、平台仓、美国仓、头程、尾程、退货、承运商和竞品能力表达。
5. YQN 观察：把前四类信号提炼成今天最值得团队统一理解的一句话。

群内只展示前三条最值得看的信号，归档页展示完整 5 模块。

筛选原则：
- 先判断今天哪条信息最影响卖家决策，不按来源平均分配。
- 老板能看方向和风险，运营能看规则变化，销售能看客户提问，内容能看选题切口，履约能看供给变化。
- 群内不放“今日动作”清单，避免把晨报写成任务派发表。

demo 模式是公开来源口径样例，不代表实时新闻。live 模式如果回退到样例数据，也必须保留测试标识。
