# YQN 北美情报晨报 V1.4 交付说明

本版本先用于本地预览和钉钉测试群；确认后可在保留现有正式群的同时新增 `YQN 直播天团`。

消息名称：YQN 跨境电商 5 分钟晨报

机器人显示名：YQN 信息小助手

副标题：美国｜墨西哥｜头程｜仓储｜配送

V1.4 已交付：
- 强制分别执行美国、墨西哥、美墨联动三组联网搜索；任一搜索失败即停止发送。
- 固定 2 条美国、2 条墨西哥、1 条美墨联动，不再用国内媒体或无关地区补位。
- 每条展示发生、生效、影响卖家、影响链路、卖家检查、YQN 可承接和官方来源。
- 卖家价值评分低于 70 分不进入群内晨报。
- 只允许已被联网结果验证的官方来源进入 live 群消息。
- 正式群与 `YQN 直播天团` 独立配置、独立记录发送结果、独立防重复。
- 06:00 北京时间预抓取 dry run，08:45 北京时间正式运行，并保留看门狗失败提醒。
- YQN 能力口径由 `knowledge/yqn-capabilities.yaml` 统一管理，不从新闻反推或编造公司能力。
- 交付包：`YQN_Daily_5_Minutes_V1_4_Delivery.zip`。

测试阶段规则：
- 标题必须显示【测试版】。
- demo 模式样例数据必须标记为测试数据；真实补充版预览同样显示【测试版】。
- 未确认前不发任何群。
- 不展示伪精确百分比置信度。
- 不展示“是否敏感：否”。
- 不展示动作清单。
