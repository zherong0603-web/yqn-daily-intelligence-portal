# YQN 跨境增长情报晨报 V1.1 交付说明

本版本用于钉钉测试群，不进入正式业务群。

消息名称：YQN 跨境增长情报晨报  
机器人建议显示名：YQN 跨境增长情报官  
副标题：市场｜平台｜竞品｜头仓配｜获客动作

V1.1 已交付：
- 五模块内容：市场与政策、平台与卖家、竞品与履约、增长与线索、YQN 今日可用动作。
- 群内短消息：只展示前 4 个信号和今日 3 个动作。
- 网页归档：展示完整 5 模块、来源、发布时间、采集时间、风险提示和归档导航。
- 发送前验收：schema、来源字段、消息长度、测试标识、敏感边界、归档链接。
- 交付包：`YQN_Crossborder_Intelligence_Brief_V1_1_Delivery.zip`。

测试阶段规则：
- 标题必须显示【测试版】。
- demo 模式样例数据必须标记为测试数据。
- 不发正式群。
- 不展示伪精确百分比置信度。
