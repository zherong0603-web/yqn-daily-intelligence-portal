# YQN 每日 5 分钟 V1.3 交付说明

本版本用于钉钉测试群，不进入正式业务群。

消息名称：YQN 每日 5 分钟  
机器人建议显示名：YQN 每日 5 分钟  
副标题：行业｜市场｜客户｜平台｜履约

V1.3 已交付：
- 五模块内容：行业市场、平台变化、客户需求、履约供给、YQN 观察。
- 群内短消息：只展示前三条最值得看的信号。
- 完整归档链接前置，方便打开网页看完整版。
- live 模式抓取真实公开 RSS / 网页候选源，不再用 demo 样例充当真实内容。
- GitHub Models / OpenAI 提炼失败时，使用真实候选源生成保守版晨报。
- 06:00 北京时间预抓取 dry-run，08:45 北京时间发送测试群。
- 内容优先级：90% 美仓 / 北美履约，10% 墨仓 / 美墨链路，并补国内跨境卖家需求。
- 网页归档：展示完整 5 模块、来源、发布时间、采集时间、风险提示和归档导航。
- 发送前验收：schema、来源字段、消息长度、测试标识、敏感边界、归档链接。
- 交付包：`YQN_Daily_5_Minutes_V1_3_Delivery.zip`。

测试阶段规则：
- 标题必须显示【测试版】。
- demo 模式样例数据必须标记为测试数据。
- 不发正式群。
- 不展示伪精确百分比置信度。
- 不展示“是否敏感：否”。
- 不展示动作清单。
