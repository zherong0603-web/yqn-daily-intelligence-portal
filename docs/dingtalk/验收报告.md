# 验收报告

## 验收范围

V1：Markdown 简洁版 + demo 模式 + 钉钉测试群。

## 已覆盖能力

- 工作日北京时间 8:45 定时触发。
- UTC cron 为 `45 0 * * 1-5`。
- 支持 `workflow_dispatch` 手动补跑。
- demo 模式不依赖 OpenAI API。
- live 模式支持 OpenAI API；没有 OpenAI API key 时支持 GitHub Models 免费限额兜底。
- 使用 zod 校验日报结构。
- 渲染钉钉 Markdown 消息。
- 支持钉钉自定义机器人 webhook。
- 支持钉钉加签 secret。
- 生成 `dist/dingtalk` 归档。
- 生成 Markdown 预览、HTML 预览、桌面截图、手机截图。
- 默认 `dry_run=true`，防止误发群。
- 命中高风险词会阻断发送并生成风险报告。

## 人工上线前检查

- 钉钉 webhook 必须是测试群。
- `dry_run=false` 前必须先看预览。
- 正式业务群不在 V1 内配置。
- 真实内部数据不进入 V1。
