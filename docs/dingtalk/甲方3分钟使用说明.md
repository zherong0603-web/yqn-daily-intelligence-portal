# 甲方 3 分钟使用说明

1. 先打开 `preview/dingtalk-markdown-preview.html`，看钉钉消息长什么样。
2. 再看 `preview/dingtalk-message-desktop.png` 和 `preview/dingtalk-message-mobile.png`，确认群内阅读是否清楚。
3. 看 `dist/dingtalk/index.html`，确认历史归档入口。
4. 第一次上线只选 `live` 和 `dry_run=true`，先看真实来源候选是否靠谱。
5. 确认测试群 webhook 正确后，再手动运行一次 `dry_run=false`。
6. 检查来源报告中美国、墨西哥、美墨联动三组搜索都为完成，且 2+2+1 配比、70 分门槛全部通过。
7. 先连续 3 个工作日发测试群，不直接进正式业务群或 `YQN 直播天团`。

判断这版是否可用，只看 5 件事：

- 标题是否醒目。
- 今日判断是否 30 字以内。
- 完整归档是否放在开头并能打开。
- 5 条是否严格为美国 2、墨西哥 2、美墨联动 1。
- 每条是否给出具体生效时间、卖家检查动作和官方来源。
