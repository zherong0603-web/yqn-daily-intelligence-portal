# 甲方 3 分钟使用说明

1. 先打开 `preview/dingtalk-markdown-preview.html`，看钉钉消息长什么样。
2. 再看 `preview/dingtalk-message-desktop.png` 和 `preview/dingtalk-message-mobile.png`，确认群内阅读是否清楚。
3. 看 `dist/dingtalk/index.html`，确认历史归档入口。
4. 第一次上线只选 `demo` 和 `dry_run=true`。
5. 确认测试群 webhook 正确后，再手动运行一次 `dry_run=false`。
6. 先连续 3 个工作日发测试群，不直接进正式业务群。

判断这版是否可用，只看 4 件事：

- 标题是否醒目。
- 今日一句话是否 30 字以内。
- 4 条信号是否能看出业务动作。
- 今日 3 个动作是否能直接分给销售、内容、履约或数据。
