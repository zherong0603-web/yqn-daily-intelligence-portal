# 安全边界

钉钉群版只发公开信息和 YQN 可公开表达的业务动作。

不能发：
- 客户名单
- 客户联系方式
- 报价
- 合同
- 毛利
- 内部成本
- 未公开客户案例
- 销售聊天记录
- 私域客户明细
- 内部项目路径
- API key
- webhook
- secret
- 个人副业
- 个人赚钱
- Codex
- OPC
- 用户个人叙事

来源边界：
- 群内 live 允许：被当次联网结果验证的政府、平台、港口、承运商等官方具体页面。
- 行业媒体只作线索，不作为重要政策的最终证据。
- 不允许：登录后台、客户表、报价表、合同、内部毛利表、销售私聊、未公开案例。

发送阻断：
1. schema 不通过，不发。
2. 来源 URL 或来源名称缺失，不发。
3. 标题缺少【测试版】，不发测试阶段消息。
4. 消息超过 2200 个非空字符，不发。
5. 出现敏感字段或禁用词，不发。
6. 归档链接无法通过 HTTP 200 校验时，不发送该链接，群内显示“归档暂未启用”。
7. 美国、墨西哥、美墨联动任一联网搜索未完成，不发。
8. 2+2+1 配比、70 分门槛、政策生效日期或官方来源校验任一失败，不发。

如果以后接入真实 MQL 表或销售反馈，必须单独做私密版。
