# 安全边界

钉钉群版只发公开信息和 YQN 可公开表达的业务动作。

不能发：
- 客户名单
- 客户联系方式
- 报价
- 合同
- 毛利
- 内部成本
- 未公开客户案例
- 销售聊天记录
- 私域客户明细
- 内部项目路径
- API key
- webhook
- secret
- 个人副业
- 个人赚钱
- Codex
- OPC
- 用户个人叙事

来源边界：
- 允许：公开官网、公开公告、公开帮助中心、公开新闻、已审核公开 FAQ。
- 不允许：登录后台、客户表、报价表、合同、内部毛利表、销售私聊、未公开案例。

发送阻断：
1. schema 不通过，不发。
2. 来源 URL 或来源名称缺失，不发。
3. 标题缺少【测试版】，不发测试阶段消息。
4. 消息超过 1200 中文字，不发。
5. 出现敏感字段或禁用词，不发。
6. 归档链接无法通过 HTTP 200 校验时，不发送该链接，群内显示“归档暂未启用”。

如果以后接入真实 MQL 表或销售反馈，必须单独做私密版。
