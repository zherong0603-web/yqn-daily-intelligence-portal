# 安全边界

群版只发公开信息和 YQN 可公开表达的业务动作。

不能发到群里的内容：

- 客户名称
- 客户联系方式
- 报价
- 合同
- 毛利
- 内部成本
- 未公开客户案例
- 销售聊天记录
- 私域客户明细
- 内部项目路径
- API key
- webhook
- secret
- 个人副业
- 个人赚钱
- 用户个人叙事

如果命中高风险词：

1. 不发送钉钉群。
2. 生成 `risk_report.json`。
3. 日志只显示风险类型，不显示原文敏感内容。
4. 如果配置了 `DINGTALK_OWNER_WEBHOOK_URL`，只给负责人发失败提醒。

如果以后要接真实 MQL 表或销售反馈，必须单独做私密版。
