# 老板演示话术

## 30 秒版本

老板，这是我搭的 YQN 每日商业情报门户。它每天自动生成增长情报、MQL 质量判断、内容实验和管理层摘要，支持历史归档、搜索、按周/月查看，也能打印成 PDF。

## 3 分钟演示路径

1. 先打开首页，看当前模式和 Demo 提醒。
2. 点“老板摘要”，展示 30 秒摘要。
3. 点“复制老板版摘要”，说明可以直接发群里。
4. 回首页搜索“美国仓”，展示历史情报检索。
5. 打开归档，展示按日期、按月、按周复盘。
6. 打开配置向导，说明真实日报只差 OpenAI API Key 和模型变量。

## 必须如实说明

当前 Demo 数据只用于验收产品形态，不代表真实每日情报。等 OpenAI API Key、OpenAI Model 配置并手动测试成功后，才能作为真实日报使用。
