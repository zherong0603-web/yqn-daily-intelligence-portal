# 视觉验收包

## 验收结论

本轮视觉验收目标是确认产品已经回到“YQN 每日重点简报”，而不是增长作战台或工程看板。

合格标准：

1. 首页首屏能看懂“今天一句话判断”。
2. 首页默认展示最多 3 条重点。
3. 每条重点都有发生了什么、为什么重要、对 YQN 的启发、建议动作、来源。
4. Demo 样例醒目标注。
5. 主导航只保留今日简报、历史简报、自动化配置、系统说明。
6. 搜索、主题筛选、历史归档、打印 PDF 都可见可用。
7. 手机端不横向滚动。
