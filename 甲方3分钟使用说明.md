# 甲方 3 分钟使用说明

## 第 1 分钟：看今日判断

打开首页：

`https://zherong0603-web.github.io/yqn-daily-intelligence-portal/`

先看首屏的“一句话判断”。如果看到 Demo 样例提示，说明这还不是正式真实 AI 简报。

## 第 2 分钟：看今日 3 个重点

往下看“今日 3 个重点”。每条重点都有：

1. 发生了什么
2. 为什么重要
3. 对 YQN 的启发
4. 建议动作
5. 来源

如果当天没有强信号，页面会直接显示“今日无强信号”。

## 第 3 分钟：复制、分享、归档

需要分享时：

1. 点“复制简报摘要”。
2. 或打开单日简报页，点“复制分享链接”。
3. 需要留档时，点“打印 / 保存 PDF”。

需要查旧内容时：

1. 点“历史简报”。
2. 按日期、月份、周查看。
3. 在首页搜索框输入关键词。
4. 用主题筛选缩小范围。

## 第一次上线要做什么

1. 打开“自动化配置”。
2. 配置 `OPENAI_API_KEY`。
3. 配置 `OPENAI_MODEL`。
4. 需要飞书通知时配置 `FEISHU_WEBHOOK_URL`。
5. 回到 Actions 手动生成一次。
