# 甲方3分钟使用说明

## 第 1 分钟：每天怎么看

1. 打开飞书卡片。
2. 点“阅读全文”。
3. 先看“管理层摘要”。
4. 再看“今天先看这 3 件事”。
5. 最后看“今日行动清单”。

## 第 2 分钟：怎么给老板看

1. 打开管理层摘要页。
2. 点“复制老板版摘要”。
3. 直接发给老板。
4. 如果老板要看完整内容，再发正式访问地址。

正式访问地址：https://zherong0603-web.github.io/yqn-daily-intelligence-portal/

## 第 3 分钟：怎么查历史

- 按日期：进归档页点某一天。
- 按月：点 `2026/07 月` 这类入口。
- 按周：点 `2026/week-27` 这类入口。
- 搜索：在首页输入关键词。
- 主题筛选：在首页选择主题。
