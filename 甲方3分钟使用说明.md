# 甲方 3 分钟使用说明

## 第 1 分钟：先看首页

打开正式网页：

https://zherong0603-web.github.io/yqn-daily-intelligence-portal/

先看四个状态：

- 当前模式：Demo 样例 / 真实日报
- OpenAI：未配置 / 已配置
- 飞书：未配置 / 已接通
- 加密：未开启 / 已开启

如果看到 Demo 提醒，不要把它当真实日报转发。

## 第 2 分钟：看老板摘要

点击“看管理层摘要”或进入：

https://zherong0603-web.github.io/yqn-daily-intelligence-portal/boss/

这里只看：

- 今日结论
- 最大异常
- 需要老板关注
- 今天推进动作
- 复制 / 打印按钮

## 第 3 分钟：打开配置向导

进入：

https://zherong0603-web.github.io/yqn-daily-intelligence-portal/setup/

按页面顺序配置：

1. OPENAI_API_KEY
2. OPENAI_MODEL
3. FEISHU_WEBHOOK_URL
4. PAGE_ACCESS_PASSPHRASE
5. 手动测试一次

密钥不要发到聊天里，只填到 GitHub Secrets。
