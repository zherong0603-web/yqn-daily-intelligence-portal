# 自动化配置说明

线上配置地址：

`https://zherong0603-web.github.io/yqn-daily-intelligence-portal/setup/`

## 页面能做什么

1. 查看 OpenAI、模型、飞书、加密是否已配置。
2. 复制 GitHub Secret / Variable 的 Name。
3. 打开 GitHub Secrets、Variables、Actions 测试入口。
4. 手动生成一次日报。

## 页面不会做什么

1. 不显示 API Key。
2. 不显示飞书 webhook。
3. 不显示页面访问密码。
4. 不要求把密钥发到聊天里。

## 最短配置路径

1. 在 GitHub Secrets 新增 `OPENAI_API_KEY`。
2. 在 GitHub Variables 新增 `OPENAI_MODEL`。
3. 需要飞书通知时，在 GitHub Secrets 新增 `FEISHU_WEBHOOK_URL`。
4. 需要加密时，在 GitHub Variables 设置 `BRIEF_ENCRYPTION_ENABLED=true`，并在 GitHub Secrets 新增 `PAGE_ACCESS_PASSPHRASE`。
5. 回到 Actions 手动运行 `Daily Briefing Portal`。
