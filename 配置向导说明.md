# 配置向导说明

## 入口

线上配置向导地址：

https://zherong0603-web.github.io/yqn-daily-intelligence-portal/setup/

## 你只差几个东西

1. `OPENAI_API_KEY`：真实 AI 日报必须。
2. `OPENAI_MODEL`：真实 AI 日报必须，必须以你 OpenAI API 账号实际可用模型为准。
3. `FEISHU_WEBHOOK_URL`：想要飞书通知时必须。
4. `PAGE_ACCESS_PASSPHRASE`：想要加密日报正文时必须。

## 页面上能看到什么

配置向导只显示“已配置 / 缺失”，不会显示真实密钥内容。

## 配错了会怎样

- API Key 错：Actions 生成日报失败。
- 模型名错：Actions 提示模型不可用。
- 飞书 webhook 错：网页能更新，但飞书收不到通知。
- 加密密码错：页面无法解锁正文。

## 不要做什么

不要把 API Key、飞书 webhook、页面密码发到聊天里。
