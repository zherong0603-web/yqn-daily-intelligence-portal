# 自动化配置说明

线上配置地址：

`https://zherong0603-web.github.io/yqn-daily-intelligence-portal/setup/`

## 页面能做什么

1. 查看 OpenAI、模型、飞书、加密是否已配置。
2. 复制 GitHub Secret / Variable 的 Name。
3. 打开 GitHub Secrets、Variables、Actions 测试入口。
4. 手动生成一次日报。

## 页面不会做什么

1. 不显示 API Key。
2. 不显示飞书 webhook。
3. 不显示页面访问密码。
4. 不要求把密钥发到聊天里。

## 最短配置路径

最傻瓜方式不是去 GitHub 设置页，而是打开本机网页配置助手：

1. 让 Codex 打开本机配置助手，或在项目里运行 `npm run setup:wizard`。
2. 浏览器会打开一个 `127.0.0.1` 开头的本机网页。
3. 填 `OPENAI_API_KEY`。
4. `OPENAI_MODEL` 默认用 `gpt-4o-mini`。
5. 需要飞书通知时，填 `FEISHU_WEBHOOK_URL`。
6. 需要加密时，勾选加密并填 `PAGE_ACCESS_PASSPHRASE`。
7. 点“保存配置”。
8. 勾选“保存后立刻跑一次真实日报测试”时，系统会自动触发 Actions。

备用方式才是手动进入 GitHub Secrets / Variables 配置。

## 为什么不能直接在公开网页保存

正式 GitHub Pages 是公开网页。公开网页如果直接保存 API Key，就会让密钥暴露，或者无法让每天自动任务安全读取。

本机配置助手只在你的电脑上运行，用当前电脑已有的 GitHub 权限写入 GitHub Secrets；密钥不会出现在公开 HTML、仓库、日志或聊天里。

## 如果想省钱

先用 `gpt-4o-mini`，不要先开搜索增强。ChatGPT Pro 会员不包含 OpenAI API 免费额度。

Google Gemini API 和阿里云 Model Studio 有免费额度路线，但当前网站还没有接入它们；要用这些路线，需要后续新增 `GEMINI_API_KEY` 或阿里云对应 API Key 的读取和生成逻辑。
