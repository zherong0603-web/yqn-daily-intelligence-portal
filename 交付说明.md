# 交付说明

## 一句话结论

V4.1 的 Demo 演示已经可用，可以打开网页、看老板摘要、查历史、搜索、打印 PDF、看配置向导。真实每日 AI 日报还需要用户本人在 GitHub 配置 OpenAI API Key 和 OpenAI Model。飞书通知需要额外配置飞书机器人。正式内部内容建议开启加密。

## 我第一次打开应该点哪里

1. 打开首页，点“打开配置向导”。
2. 按配置向导补齐 OpenAI / 飞书 / 加密。
3. 回到 Actions，手动测试一次 Daily Briefing Portal。

## 每天怎么用

1. 先看飞书入口卡片。
2. 点“阅读全文”打开网页。
3. 先看“老板 30 秒摘要”。
4. 再看“今日行动清单”。
5. 需要复盘时查历史归档，或点击“打印 / 保存 PDF”。

## 配置 API：像教小白一样

1. 打开 https://platform.openai.com/api-keys。
2. 登录你的 OpenAI 账号。
3. 点击 Create new secret key。
4. 创建后复制那串 API Key。不要发给任何人，不要发到聊天里。
5. 打开 GitHub 仓库的 Settings。
6. 点击 Secrets and variables。
7. 点击 Actions。
8. 进入 Secrets。
9. 点击 New repository secret。
10. Name 填 `OPENAI_API_KEY`。
11. Value 粘贴你的 OpenAI API Key。
12. 点击 Add secret 保存。
13. 保存后去 Actions 手动运行一次。
14. 如果失败，打开失败的 step 截图，不要截图密钥。

## 配置模型：像教小白一样

1. `OPENAI_MODEL` 是告诉系统用哪个 OpenAI API 模型生成日报。
2. 不要乱填模型名，也不要把未经确认的模型写死成唯一默认值。
3. 打开 OpenAI 模型文档，确认你的 API 账号可用模型。
4. 回到 GitHub 仓库 Settings。
5. 点击 Secrets and variables。
6. 点击 Actions。
7. 进入 Variables。
8. 点击 New repository variable。
9. Name 填 `OPENAI_MODEL`。
10. Value 填你账号实际可用的模型名，优先选择可用的低成本 mini 模型。
11. 点击 Add variable 保存。
12. 配错时，Actions 会提示模型不可用。

## 配置飞书：像教小白一样

1. 打开要接收通知的飞书群。
2. 进入群设置。
3. 找到机器人。
4. 添加自定义机器人。
5. 复制 webhook 地址。
6. 打开 GitHub 仓库 Settings → Secrets and variables → Actions → Secrets。
7. 点击 New repository secret。
8. Name 填 `FEISHU_WEBHOOK_URL`。
9. Value 粘贴飞书 webhook。
10. 点击 Add secret 保存。
11. 配完后手动跑一次 Actions。
12. 网页更新但飞书没收到，优先检查 webhook 是否复制完整。

## 配置加密：像教小白一样

1. GitHub Pages 是公开网页，正式内容建议开启加密。
2. 打开 GitHub 仓库 Settings → Secrets and variables → Actions → Variables。
3. 新增或修改 `BRIEF_ENCRYPTION_ENABLED`，Value 填 `true`。
4. 打开 Secrets。
5. 新增 `PAGE_ACCESS_PASSPHRASE`。
6. Value 填你自己设置的强密码。
7. 保存后手动跑一次 Actions。
8. 页面打开后输入密码解锁日报正文。
9. 密码忘了就重新设置 `PAGE_ACCESS_PASSPHRASE`，再跑一次 Actions。

## 安全边界

- GitHub Pages 是公开网页。
- noindex / robots 不是访问控制。
- 加密模式不是企业级登录系统。
- 不要放客户名单、报价、合同、内部成本、密钥、私密线索。
- 不要把 API Key、飞书 webhook、页面密码发到聊天里。

## 成本边界

- ChatGPT Pro 不等于 OpenAI API 免费额度。
- OpenAI API 单独按量计费。
- GitHub Pages / Actions 当前方案尽量免费。
- 搜索增强可能产生额外费用；当前默认关闭搜索增强。
